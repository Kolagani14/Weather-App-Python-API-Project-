# 🌦️ Weather App (Python API Project)

A command-line weather application built using Python that fetches real-time weather data from an external API. The program allows users to enter a city name and retrieve current temperature and weather conditions.

## 🚀 Features
- Fetch real-time weather data using API
- Displays temperature in Celsius
- Shows weather description (e.g., clear sky, rain)
- Handles invalid city names and API errors
- Uses HTTP requests and JSON data parsing

## 🛠️ Technologies Used
- Python
- requests library
- OpenWeatherMap API

## ▶️ How It Works
1. User enters a city name
2. Program sends a request to the weather API
3. API returns weather data in JSON format
4. Program extracts temperature and weather details
5. Displays formatted output to the user

## 💻 Example Output
- Enter city name: guntur
- Status Code: 200
- City: guntur
- Temperature: 32°C
- Weather: clear sky
